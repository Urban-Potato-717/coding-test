# 올림 나눗셈

## 한 줄 설명

나누어떨어지지 않을 때 몫을 하나 올려, 필요한 묶음의 최소 개수를 구하는 계산이다.

## 언제 떠올리는가

- “남는 것이 있으면 하나가 더 필요하다”는 문제
- 피자 판 수, 버스 수, 페이지 수처럼 일부만 사용해도 한 단위 전체가 필요한 경우
- `n`개를 한 묶음에 `size`개씩 담을 때 필요한 묶음 수

## 내가 헷갈린 부분

`//`는 결과를 내림하므로 나머지가 있는 경우를 표현하지 못한다.

```python
8 // 7  # 1
```

7명까지 담은 뒤 1명이 남으므로 실제로는 2묶음이 필요하다. 나머지가 있는 경우에만 1을 더해야 한다.

## 최소 예제

```python
def ceiling_division(n, size):
    if n % size == 0:
        return n // size
    return n // size + 1
```

같은 계산을 한 줄로 표현할 수도 있다.

```python
def ceiling_division(n, size):
    return (n + size - 1) // size
```

양의 정수 `n`에는 `(n - 1) // size + 1`도 같은 결과를 낸다.

## 값으로 확인하기

```text
n=7, size=7  -> 1
n=8, size=7  -> 2
n=14, size=7 -> 2
n=15, size=7 -> 3
```

- `programmers/0626_pizza(3).py`: 한 판의 조각 수가 달라지는 경우
