# 리스트 VS 딕셔너리 (최빈값 구하기)

1. 리스트 (List)
**순서가 있는 값들의 모음**

```python
  array = [10, 20, 30, 40]
           0   1   2   3  ← 인덱스 (자동 부여)
```
- 값에 접근할 때 순서(인덱스)로 찾음
- array[2] -> 30
- 인덱스는 항상 **정수** 만 가능

2. 딕셔너리 (Dictionary)
**내가 정한 key로 값을 지정하는 모음**

```python
  freq = {"apple": 3, "banana": 1, "cherry": 3}
            key : value
```

- 값에 접근할 때 key로 찾음
- freq["apple"] -> 3
- key는 숫자, 문자열 등 뭐든 가능

+ 딕셔너리명.keys() 처럼 추가로 뽑아낼 수 있는 기능
- keys() → key만 필요할 때 (숫자 목록)
- values() → value만 필요할 때 (횟수 목록)
- items() → 둘 다 필요할 때 (modes 구할 때처럼)
