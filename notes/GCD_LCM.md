# 최대공약수와 최소공배수

## 한 줄 설명

- 최대공약수(GCD): 두 수를 모두 나눌 수 있는 가장 큰 수
- 최소공배수(LCM): 두 수의 배수 중 가장 작은 공통 값

## 언제 떠올리는가

### GCD

- 분수를 가장 간단한 형태로 약분할 때
- 여러 수를 같은 크기로 최대한 크게 나눌 때

### LCM

- 두 조건을 동시에 만족하는 가장 작은 수를 구할 때
- 서로 다른 주기가 다시 만나는 최초 시점을 구할 때
- 여러 사람에게 같은 개수로 남김없이 나눌 총량을 구할 때

## 내가 헷갈린 부분

유클리드 호제법에서 `a`와 `b`가 계속 바뀌는 이유가 헷갈렸다. `b`가 0이 되면 `a`가 마지막으로 나누어떨어지게 만든 수이므로 최대공약수다.

## 유클리드 호제법 최소 예제

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a
```

`GCD(10, 8)`의 변화:

```text
(10, 8) -> (8, 2)
(8, 2)  -> (2, 0)
GCD = 2
```

오른쪽의 `b, a % b`를 모두 계산한 후 왼쪽의 `a, b`에 한 번에 대입한다.

## 최소공배수

```python
def lcm(a, b):
    return a * b // gcd(a, b)
```

6조각짜리 피자를 10명이 같은 수만큼 남김없이 먹으려면 총 조각 수는 6과 10의 공통 배수여야 한다.

```text
6의 배수:  6, 12, 18, 24, 30, ...
10의 배수: 10, 20, 30, ...
가장 작은 공통 값: 30
필요한 피자: 30 // 6 = 5판
```

## 파이썬 내장 함수

```python
import math

math.gcd(10, 8)  # 2
math.lcm(6, 10)  # 30
```
