# 유클리드 호제법

두 수의 최대공약수(GCD)를 구하는 알고리즘.
두 수가 공통으로 나누어 떨어지는 수 중 가장 큰 수.
큰 수를 작은 수로 나눈 **나머지**로 계속 반복하다가, 나머지가 0이 되면 그때의 나누는 수가 GCD.

**예시:** GCD(10, 8)
10 % 8 = 2  → (8, 2)
 8 % 2 = 0  → 나머지 0, GCD = 2

## 공식
```python
def Euclidean(a, b):
    while b != 0:         # 나머지가 0이 될 때까지 반복
        a, b = b, a % b   # (a, b) → (b, a%b) 로 교체
    return a              # b가 0이 되면 a가 최대공약수
```

# 최소공배수 (LCM)

두 수가 공통으로 가지는 배수 중 가장 작은 수.

## 언제 LCM을 떠올리나?

"두 조건을 동시에 만족하는 가장 작은 수"를 찾을 때.

예: 10명이 6조각짜리 피자를 남김없이 똑같이 나눠야 할 때
- 총 조각 수가 **6의 배수**여야 함 (피자 한 판이 6조각)
- 총 조각 수가 **10의 배수**여야 함 (10명이 똑같이 나눠야 함)

```
6의 배수:  6, 12, 18, 24, 30 ...
10의 배수: 10, 20, 30 ...

둘 다 해당: 30 → 30 ÷ 6 = 5판
```

공식을 외워서 떠올리는 게 아니라, **두 조건을 동시에 만족하는 최솟값을 찾다 보면 자연스럽게 LCM이 나온다.**

**예시:** LCM(6, 10)
- 6의 배수: 6, 12, 18, 24, **30**, ...
- 10의 배수: 10, 20, **30**, ...
- LCM = 30

## 공식

GCD를 알면 LCM은 바로 구할 수 있어.


LCM(a, b) = a * b // GCD(a, b) **두 수 곱하기 // 두 수의 최대 공약수**

**예시:** LCM(6, 10)
- GCD(6, 10) = 2
- LCM = 6 * 10 // 2 = 30

왜 이 공식이 성립하냐면, 두 수를 곱하면 GCD가 두 번 들어가 있어서 한 번 나눠주는 거야.

```python
def lcm(a, b):
    return a * b // Euclidean(a, b)
```

## 파이썬 내장 함수

```python
import math
math.lcm(6, 10)  # 30
```
