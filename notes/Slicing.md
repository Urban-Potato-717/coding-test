# 슬라이싱 (Slicing)

```python
list[start:stop:step]
#    시작  :  끝  : 간격
```
`start`, `stop`, `step` 모두 생략 가능. 생략 시 기본값 적용.
`stop`은 항상 **미포함**.

---

## 기본 사용

```python
nums = [0, 1, 2, 3, 4, 5]

nums[1:4]    # → [1, 2, 3]      start=1, stop=4 (4 미포함)
nums[2:]     # → [2, 3, 4, 5]   stop 생략 = 끝까지
nums[:3]     # → [0, 1, 2]      start 생략 = 처음부터
nums[::2]    # → [0, 2, 4]      step=2, 하나 건너뛰기
```

## 음수 step - 뒤에서부터

```python
nums[::-1]    # → [5, 4, 3, 2, 1, 0]   전체 뒤집기
nums[::-2]    # → [5, 3, 1]             뒤에서 하나 건너뛰기
nums[4:1:-1]  # → [4, 3, 2]             index 4→2 역방향
```

## 음수 인덱스

```python
nums[-1]     # → 5              마지막 요소
nums[-2]     # → 4              뒤에서 두 번째
nums[-3:]    # → [3, 4, 5]      뒤에서 3개
nums[:-2]    # → [0, 1, 2, 3]   마지막 2개 제외
```

---

## 자주 쓰는 패턴

| 패턴 | 의미 |
|------|------|
| `[:]` | 전체 복사 |
| `[::-1]` | 전체 뒤집기 |
| `[::2]` | 짝수 인덱스만 |
| `[1::2]` | 홀수 인덱스만 |
| `[-n:]` | 마지막 n개 |
| `[:-n]` | 마지막 n개 제외 |
