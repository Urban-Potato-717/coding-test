# 슬라이싱

## 한 줄 설명

문자열이나 리스트의 일부를 `start:stop:step` 규칙으로 잘라 새로운 값을 만드는 문법이다.

## 언제 떠올리는가

- 일부 구간이나 마지막 몇 개의 값이 필요할 때
- 원본 순서를 뒤집을 때
- 일정한 간격으로 값을 선택할 때

## 내가 헷갈린 부분

`stop` 위치는 결과에 포함되지 않는다. 또한 `sort(reverse=True)`는 원본 순서를 뒤집는 것이 아니라 값을 내림차순 정렬한다.

## 최소 예제

```python
numbers = [0, 1, 2, 3, 4, 5]

numbers[1:4]   # [1, 2, 3]
numbers[2:]    # [2, 3, 4, 5]
numbers[:3]    # [0, 1, 2]
numbers[::2]   # [0, 2, 4]
numbers[::-1]  # [5, 4, 3, 2, 1, 0]
"python"[::-1]  # "nohtyp"
```

## 자주 쓰는 패턴

| 패턴 | 의미 |
|---|---|
| `[:]` | 전체 복사 |
| `[::-1]` | 전체 뒤집기 |
| `[::2]` | 짝수 인덱스 선택 |
| `[1::2]` | 홀수 인덱스 선택 |
| `[-n:]` | 마지막 n개 |
| `[:-n]` | 마지막 n개 제외 |

## 뒤집기와 정렬의 차이

```python
numbers = [2, 1, 3]

numbers[::-1]                  # [3, 1, 2]
sorted(numbers, reverse=True)  # [3, 2, 1]
```
