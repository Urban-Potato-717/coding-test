# 유클리드 호제법

두 수의 최대공약수(GCD)를 구하는 알고리즘.
큰 수를 작은 수로 나눈 **나머지**로 계속 반복하다가, 나머지가 0이 되면 그때의 나누는 수가 GCD.

**예시:** GCD(10, 8)
10 % 8 = 2  → (8, 2)
 8 % 2 = 0  → 나머지 0, GCD = 2

```python
def Euclidean(a, b):
    while b != 0:         # 나머지가 0이 될 때까지 반복
        a, b = b, a % b   # (a, b) → (b, a%b) 로 교체
    return a              # b가 0이 되면 a가 최대공약수
```
